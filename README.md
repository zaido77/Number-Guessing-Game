# Number Guessing Game

A simple **Windows Forms (WinForms) game** developed in C# where the player tries to guess a randomly generated number. The game includes **three difficulty modes**: Easy, Medium, and Hard.

## ğŸ® Features

- **Three difficulty modes**:
  - **Easy**: Guess between 1 - 5
  - **Medium**: Guess between 1 - 10
  - **Hard**: Guess between 1 - 20
- **Interactive WinForms UI**
  - Input field for guesses
  - Buttons to submit guess and restart the game
